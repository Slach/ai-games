# Правила игры AI Game Master

## Обзор игры

AI Game Master — это кооперативная игра с генерируемым искусственным интеллектом сюжетом, доставляемая через Telegram-бот. Каждый день генерируется уникальная история в космическом сеттинге (звёздный корабль, экипаж, приключения), где игроки принимают решения, влияющие на развитие сюжета.

## Основные механики игры

### 1. Регистрация и онбординг

- После начала сессии (`/start`) игрок проходит интервью для определения своей роли
- Интервью состоит из вопросов типа "что бы вы сделали в этой ситуации" с вариантами действий
- Каждый вариант ответа содержит очки (role_scores) для разных ролей экипажа
- Количество вопросов и вариантов настраивается через ONBOARDING_QUESTIONS_COUNT и ONBOARDING_OPTIONS_COUNT в .env
- На основе ответов игрока через comfyui генерируется персональный аватар
- Аватар и описание роли отправляются игроку после завершения онбординга

### 2. Формирование экипажа

- Несколько игр могут запускаться одновременно для разных групп игроков
- Игрок может участвовать только в одной игре одновременно
- Экипаж собирается в течение 3 дней с момента присоединения первого члена команды
- Если команда не полна к концу срока, незаполненные роли назначаются NPC ботам
- Каждый игрок получает уникальную роль (Chief Engineer, XO, Science Officer и т.д.)

### 3. Игровой цикл

Каждый игровой день проходит по следующему сценарию:

```
┌─────────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────────┤
```

## Поведение Game Master (корабельный компьютер)

### При начале нового игрового дня:

1. **Проверка состояния игры**
   - Game Master API проверяет последнее состояние игры
   - Проверяет, что игра не закончена (корабль и экипаж живы)
   - Если игра закончена, генерация прекращается

2. **Генерация сюжета**
   - Генерирует вступительную историю для нового дня
   - Использует действия игроков из предыдущего дня
   - Адаптирует сюжет с учётом последствий выборов
   - Обеспечивает последовательность и связность истории

3. **Генерация контента**
   - Через comfyui генерирует персонализированные комиксы
   - Каждый игрок получает свою историю согласно роли
   - Все истории объединены общей вступительной историей дня

4. **Обработка действий игроков**
   - Игрок выбирает действие из предложенных вариантов
   - Последствия записываются как вход для следующей генерации
   - Если игрок не выбрал действие вовремя, AI Game Master выбирает за них

## Взаимодействие с игроками

### Текстовые сообщения:

- Игроки могут общаться с Game Master через текстовые сообщения
- Сообщения используются для генерации текста и ответов
- Сообщения передаются другим игрокам вместе с аватаром автора
- Game Master отвечает в характере корабельного компьютера

### Голосовые сообщения:

- Поддержка голосовых сообщений (без транскрибации)
- Запись и хранение голосовых сообщений
- Обработка через Game Master API

## Роли игроков

| Роль | Описание | Характеристики |
|------|----------|----------------|
| Chief Engineer | Отвечает за техническое состояние корабля | Технический, практичный, решительный |
| XO (First Officer) | Координирует действия экипажа | Коммуникабельный, стратегический, эмпатичный |
| Science Officer | Исследует неизвестное и анализирует данные | Аналитический, любопытный, методичный |
| Security Chief | Обеспечивает безопасность экипажа | Бдительный, защитник, скептик |
| Pilot | Навигатор корабля | Приключенческий, быстрый, интуитивный |

## Игровые механики

### Действия и последствия:

- Каждое действие имеет видимый выбор и скрытое последствие
- Последствия влияют на развитие сюжета в следующих днях
- Система последствий создаёт ощущение реального влияния на мир игры

### Командная игра:

- Решения принимаются группой игроков
- Влияют на общую историю
- Создаётся чувство сообщества и совместного приключения

## Часто задаваемые вопросы

**Q: Как часто генерируется новый сюжет?**
A: Каждый день в назначенное время (по умолчанию 08:00)

**Q: Что если я пропущу выбор действия?**
A: AI Game Master автоматически выберет за вас на основе вашей роли и характеристик

**Q: Могу ли я участвовать в нескольких играх одновременно?**
A: Нет, игрок может участвовать только в одной игре одновременно

**Q: Как долго длится одна игра?**
A: Игра продолжается до тех пор, пока корабль и экипаж живы. Финал зависит от решений игроков

**Q: Можно ли общаться с другими игроками?**
A: Сообщения передаются через Game Master и видны другим игрокам вместе с аватарами

## Заключение

AI Game Master создаёт уникальный ежедневный опыт, где каждый игрок становится частью живой истории. Генерируемый контент, персональные комиксы и кооперативная механика создают ощущение настоящего космического приключения.
